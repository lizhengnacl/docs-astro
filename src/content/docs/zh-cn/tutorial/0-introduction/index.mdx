<Lede> 在本教程中，您将通过从零开始构建一个完全功能的博客来学习Astro的关键特性，直到正式发布！ 🚀 </Lede>

沿途，你会：
- 设置您的开发环境
- 为您的网站创建页面和博客文章
- 使用Astro组件构建
- 查询和处理本地文件
- 为您的网站添加交互性
- 将您的网站部署到网络上

想要预览你将要构建的内容吗？您可以在[GitHub](https://github.com/withastro/blog-tutorial-demo)或[StackBlitz](https://stackblitz.com/github/withastro/blog-tutorial-demo/tree/complete?file=src%2Fpages%2Findex.astro)上查看最终项目。

:::note
如果您更愿意使用预先构建的Astro网站开始探索，可以访问 https://astro.new 并选择一个起始模板在线上编辑器中打开和编辑。
:::
